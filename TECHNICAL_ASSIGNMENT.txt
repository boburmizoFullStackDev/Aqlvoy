1. PROJECT OVERVIEW
================================================================================

AQLVOY is a full-stack web application designed to provide intelligent tutoring
services through an AI chatbot interface. The platform supports three different
user roles (Student, Teacher, Parent) with dedicated dashboards and personalized
learning experiences.

Language: Uzbek (O'zbek tili)
Platform: Web-based (responsive design for desktop and mobile)
Status: Development with Django development server running

7. API INTEGRATION (Backend Communication)
================================================================================

API Handler: app-api.js

Key Functions:
  - apiRequest(path, options)
    * Handles all HTTP requests to backend
    * Base URL: window.location.origin (dynamic)
    * Content-Type: application/json
    * Error handling with user-friendly messages
  
  - persistUserSession(user)
    * Saves user data to localStorage
    * Stores role-specific information
    * Maintains session across page refreshes
  
  - routeForRole(role)
    * Returns appropriate dashboard page based on user role
    * student → student.html
    * teacher → teacher.html
    * parent → parent.html
  
  - currentUserEmail(fallback)
    * Retrieves user's email from localStorage
    * Returns fallback value if not found

Expected Backend Endpoints (to be implemented):
  - POST /login - User authentication
  - POST /register - New user registration
  - GET /user/profile - Get user information
  - POST /logout - End user session
  - POST /chat - Send message to AI
  - GET /progress - Retrieve learning progress
  - POST /tests/submit - Submit test answers
